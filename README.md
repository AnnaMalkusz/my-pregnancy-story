
# My Pregnancy Story

> **Note:** This repository contains the **Portfolio Edition** of the application, prepared for demonstration and recruitment purposes. It showcases the complete pregnancy planning workflow and key features while using fictional data to protect personal information.

*A modern pregnancy companion designed to help expectant parents organize appointments, documents, shopping lists, memories and important milestones in one beautiful workspace.*

---

# 🚀 Live Demo

👉 **[Explore Demo Application](https://annamalkusz.github.io/my-pregnancy-story/)**

---

# ✨ Features

- Pregnancy timeline & due date tracking
- Calendar for appointments and important events
- Shopping lists with budget tracking
- Medical document management
- Pregnancy journal & milestone memories
- Belly & ultrasound photo gallery
- Healthcare contacts management
- Local-first architecture
